3. 404 页面

访问不存在的路径时（如 /abc），GitHub Pages 默认白屏。加一个 404 页面：

404.html
页面效果：友好的企鹅提示 + "这个页面飘走了 🏝️" + 返回首页按钮

好处：访问者不会看到 GitHub 默认错误页，保持品牌一致性

4. robots.txt & sitemap.xml

robots.txt
User-agent: *
Allow: /
Sitemap: https://yuanxuru-oss.github.io/Ayu-websites-pages/ayu-site/sitemap.xml
sitemap.xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://yuanxuru-oss.github.io/Ayu-websites-pages/ayu-site/</loc>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://yuanxuru-oss.github.io/Ayu-websites-pages/ayu-site/admin.html</loc>
    <priority>0.1</priority>
  </url>
</urlset>

好处：告诉 Google/Bing 你的网站结构，帮助搜索引擎收录
